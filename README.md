# Student Performance Predictor

A beginner machine learning project built using Python and Scikit-learn.

## Features
- Predicts student pass/fail
- Uses Decision Tree Classifier
- Feature scaling using StandardScaler
- Calculates accuracy and confusion matrix
- Visualizes data using Matplotlib
- Saves trained model using Joblib
